# [IT's Tinkoff Cup 2023 Frontend](https://www.tinkoff.ru/solutioncup/frontend/)
## Getting Started

First, run the development server:

```bash
npm install
npm run "dev and server"
```
```bash
yarn install
yarn run "dev and server"
```
